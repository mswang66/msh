# msh
该项目主要是为了在mac和linux下纯命令行操作的时候，没有xshell之类的客户端，可以用来管理远程连接列表。


### 用法
##### 1. 列表
```
msh -l
 ------- ----------------- ---------------
| index | Host            | UserName      |
 ------- ----------------- ---------------
| 1     | 192.168.122.45  | root          |
 ------- ----------------- ---------------
| 2     | 192.168.200.249 | root          |
 ------- ----------------- ---------------
| 3     | 192.168.200.242 | root          |
 ------- ----------------- ---------------
| 4     | 192.168.200.250 | root          |
 ------- ----------------- ---------------
| 5     | 192.168.2.72    | root          |
 ------- ----------------- ---------------
 ```
 
 
 ##### 2. 添加
 ```
 msh -a
 ```
 按照提示输入即可
